- **Note**:
  - only the basic APIs in `data_juicer` and two basic tools
    (data `processing <quick-start.html#data-processing>`_ and `analysis <quick-start.html#data-analysis>`_) are available in this way. If you want customizable
    and complete functions, we recommend you install `data_juicer` `from source <#from-Source-for-specific-scenarios>`_.
  - The release versions from pypi have a certain lag compared to the latest version from source.
    So if you want to follow the latest functions of `data_juicer`, we recommend you install `from source <#from-Source-for-specific-scenarios>`_.
