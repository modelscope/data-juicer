# clean_copyright_mapper

Cleans copyright comments at the beginning of text samples.

This operator removes copyright comments from the start of text samples. It identifies
and strips multiline comments that contain the word "copyright" using a regular
expression. It also greedily removes lines starting with comment markers like `//`, `#`,
or `--` at the beginning of the text, as these are often part of copyright headers. The
operator processes each sample individually but can handle batches for efficiency.

Type 算子类型: **mapper**

Tags 标签: cpu, text

#### ✨ explanation 解释
This example demonstrates how the operator removes copyright comments and specific comment lines at the start of text. For instance, it deletes the multiline comment containing 'copyright' in the first sample, as well as all leading lines starting with `//`, `#`, or `--` in other samples. However, if the comment does not contain 'copyright', like in the second sample, it is kept. The final output shows the cleaned text after removing these elements. Note that the output data directly reflects the result of the operator's processing, without any further modifications.
这个例子展示了算子如何移除文本开头的版权注释和特定的注释行。例如，它会删除第一个样本中包含'copyright'的多行注释，以及其它样本中以`//`、`#`或`--`开始的所有前置行。但是，如果注释不包含'copyright'，如第二个样本所示，则保留该注释。最终输出显示了在移除这些元素后的清理文本。请注意，输出数据直接反映了算子处理的结果，没有进行任何进一步修改。
