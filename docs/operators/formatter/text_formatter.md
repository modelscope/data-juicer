# text_formatter

The class is used to load and format text-type files.

e.g. `['.txt', '.pdf', '.cpp', '.docx']`

类用于加载和格式化文本类型文件。

例如 `'.txt '，'.pdf'，'.Cpp'，'.docx'`

Type 算子类型: **formatter**

Tags 标签: cpu, text
